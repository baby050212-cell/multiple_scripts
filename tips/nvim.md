## Vim / Neovim 进阶效率清单（开发级）


### 🚀 高频移动（替代方向键）

| 功能 | 快捷键 | 说明 |
|------|--------|------|
| 下一个单词开头 | w | 快速向前跳词 |
| 上一个单词开头 | b | 向后跳词 |
| 单词结尾 | e | 跳到词尾 |
| 行首 | 0 | 第一列 |
| 首个非空字符 | ^ | 跳过缩进 |
| 行尾 | $ | 行尾 |
| 最后非空字符 | g_ | 常用于复制 |

---

### 📏 相对行号跳转（补充）

| 功能 | 快捷键 | 说明 |
|------|--------|------|
| 向下跳 N 行 | Nj | 例如 5j：向下跳 5 行 |
| 向上跳 N 行 | Nk | 例如 8k：向上跳 8 行 |
| 删除到下方 N 行 | dNj | 例如 d5j：删除到下方第 5 行 |
| 复制到下方 N 行 | yNj | 例如 y4j：复制到下方第 4 行 |
| 可视选中下方 N 行 | VNj | 例如 V6j：选中当前行到下方第 6 行 |

---

### 🧠 记忆法

左侧相对行号显示几，就用：

text 数字 + j/k

例如左侧显示 12，要跳过去就是：

vim 12j

---

### 🎯 快速修改（最提升效率）

| 功能 | 快捷键 | 说明 |
|------|--------|------|
| 改一个单词 | ciw | change inner word |
| 改整行 | cc | 删除并进入插入 |
| 改到行尾 | c$ | 从光标到行尾 |
| 改括号内 | ci( | 不含括号 |
| 改引号内 | ci" | 不含引号 |
| 删除单词 | diw | delete inner word |
| 删除括号内容 | di( | 不含括号 |

---

### 📦 复制 / 删除范围（文本对象）

| 功能 | 快捷键 | 说明 |
|------|--------|------|
| 复制单词 | yiw | yank inner word |
| 复制括号内 | yi( | 不含括号 |
| 复制括号含边界 | ya( | 包含括号 |
| 删除引号内 | di" | 常用于字符串 |
| 删除标签内 | dit | HTML/XML |
| 复制段落 | yap | paragraph |

### ⚡ 实战组合（开发高频）

| 场景 | 快捷组合 | 说明 |
|------|-----------|------|
| 改函数参数 | f( → ci( | 高效 |
| 改字符串 | ci" | 常用 |
| 删除整段逻辑 | dap | 删除段落 |
| 快速复制变量 | yiw → p | 复制粘贴 |
| 多处修改 | ciw → . | 批量修改 |
| 复制整行无换行 | 0vg_y | 结合上一表 |

---

### 🧩 核心思维（关键）

- 动作：c / d / y
- 范围：iw / ( / " / p
- 组合：动作 + 范围

例如：

text ciw = 改单词 di( = 删除括号内 yi" = 复制引号内容

---

### ✅ 一句话总结

> 👉 Vim 的本质 = 动作（操作） + 范围（对象），熟练后几乎不用方向键。
