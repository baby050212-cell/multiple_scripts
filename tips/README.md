# tips

个人配置和速查资料。

这些文件主要用于参考、查找和复制片段，不建议直接全量覆盖本机配置。

## 文件

- `init.lua`：Neovim 基础配置示例，包括 UI、缩进、搜索、剪贴板、快捷键和自动命令。
- `nvim.md`：Vim/Neovim 高频操作速查。
- `zshrc`：zsh 配置示例，包括 Homebrew、编辑器、Python、Node、代理和 Claude Code 相关环境变量。

## 使用建议

复制前按本机环境调整：

- 本机路径
- shell 工具安装位置
- 代理端口
- API key、token 等敏感值
- 默认模型或服务地址

真实密钥不要提交到仓库，建议使用占位符或放在本机私有配置中。
